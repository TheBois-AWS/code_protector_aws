Getting Started

This project runs a static frontend behind CloudFront, and a Lambda backend with DynamoDB + S3. Documentation is served under /docs from the same frontend bucket.

Prerequisites

  • Node.js 20+
  • AWS CLI + SAM CLI for infrastructure deployment
  • GitHub Actions OIDC role for CI deploy (optional but recommended)

Install

npm install

Run Local API

npm run dev
# API: http://localhost:3001

Run Docs Host

For docs screenshots and preview, use the docs host script that serves frontend/ and proxies /api and /files to a local deterministic mock server. 

npm run docs:host

Generate Screenshots

npm run docs:capture

Validate API Contract

npm run docs:validate-openapi
npm run docs:check-parity
npm run docs:check

Keep frontend/docs/openapi.yaml aligned with src/router.js. CI will fail if route parity drifts.
